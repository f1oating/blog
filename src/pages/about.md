I'm **Dmytro Toronchenko**, a graphics programmer focused on real-time rendering. I work with Vulkan, DX12, and C++, and I care about the low-level details that turn draw calls into images on screen.

This blog is where I write about things I figure out — render pass structures, memory aliasing, synchronization, shader optimization, and whatever else comes up while building a renderer.
